loewenrudel.net
===============

Install:

  * gem install jekyll

Use:

  * jekyll serve --watch
  * visit 127.0.0.1:4000

New Post:

  * rake post title="Hello World"

Relevant Files:

  * index.md
  * _includes/themes/twitter/default.html (html template)
  * _includes/themes/twitter/post.html (html template)
  * assets/themes/twitter/css/style.css

1 - Entscheidung benennen

Auf dieser Seite gibt der User lediglich die Entscheidung an. In einem Hilfetext können wir ggf. darauf hinweisen, wie die Entscheidung sinnvoll formuliert sein sollte - also z.B. nicht als Frage.

2 - Optionen finden

Auf dieser Seite gibt der User die bereits bekannten Optionen ein.  [+] erstellt eine neue Option

3 - Vorhandenes Wissen

Sind die Optionen eingetragen, ändert sich das Aussehen der Seite: Die Optionen werden "zusammengeschoben" und neben der jeweiligen Option können die bekannten Fakten zu der Option eingetragen werden.

Im ersten Schritt werden die bekannten Fakten eingetragen ("Eigenes Wissen"). Ist das passiert, wird diese Seite nach einem Klick auf "Wissen vertiefen" nach unten verlängert und die nächste Seite wird angehängt.

4 - Wissen vertiefen

Jeder der sechs Bereiche öffnet einen Hilfetext und schlägt konkrete Quellen vor. Mit Hilfe der Quellen (der User verlässt hier die Seite) lassen sich Informationen finden, die ganz unten auf der Seite eingetragen werden können.

Beim Eintragen von Informationen wird die Aussage in das Textfeld eingetragen. Ein Klick auf das [+] öffnet das nächste Formular, in dem die Information einer Option zugeordnet wird.

5 - Wissen sammeln

6 - Gesammeltes Wissen

Das Sammeln von Wissen ergänzt die Zeilen zu den Optionen, die bereits mit dem eigenen Wissen eingetragen wurden.

Ist der User der Meinung dass er genügend Informationen zusammengetragen hat, verändert sich durch einen Klick auf "Bewerte die Optionen" die Ansicht der Seite erneut:

7 - Optionen bewerten

Die Darstellung der gefundenen Informationen wird konmprimiert und neben jeder Option erscheint ein "Bewerten" Button. Durch einen Klick auf den Button steigt man in die Bewertung der jeweiligen Option ein.

Ist eine Option bewertet, wird die Bewertung statt des Buttons gezeigt.

8 - Bewertung

9 - Optionen bewertet

Sind alle Optionen bewertet, kann die Entscheidung getroffen werden.
